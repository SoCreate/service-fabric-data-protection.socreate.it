Setup the Data Protection Store

Requirements: Use within a Service Fabric Cluster
First, create a new .NET Core Stateful Service. Next, since Data Protection uses the Service Fabric Distributed Cache Store to persist its keys, you'll need to install the Nuget package: ``` dotnet add package SoCreate.Extensions.Caching.ServiceFabric ``` Extend the new stateful service using "DistributedCacheStoreService" as the base class.
